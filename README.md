# X Tools (no more Twitter)

## What can I do?

- Delete Tweets
- Delete Replies
- Remove Likes
- Remove Followings
- Delete Media Tweet

#### Installation

```sh
git clone https://github.com/aka-deVOID/x-tools.git
cd x-tools
pip install -r requirements.txt
python src/main.py
```
